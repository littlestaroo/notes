# 2.1名称

Go 中函数、变量、常量、类型、语句标签和包的名称遵循一个简单的规则：名称的开头是一个字母（Unicode 中的字符即可）或下划线，后面可以跟任意数量的字符、数字和下划线，并区分大小写。如 heapSort 和 HeapSort 是不同的名称。

Go 有 25 个像 if 和 switch 这样的关键字，只能用在语法允许的地方，它们不能作为名称。
